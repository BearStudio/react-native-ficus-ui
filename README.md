# react-native-ficus-ui

React Native UI library inspired by Magnus UI and Chakra UI

## Installation

```sh
npm install react-native-ficus-ui
```

## Usage

```js
import { multiply } from 'react-native-ficus-ui';

// ...

const result = await multiply(3, 7);
```
