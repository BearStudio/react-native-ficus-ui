# Spinner

Simple spinner component that is based on react native `ActivityIndicator` component.

## Import

```js
import { Spinner } from "react-native-ficus-ui";
```

## Usage

<CodeEditor code={`<HStack spacing={10}>
  <Spinner />
  <Spinner color="red" size="large" />
  <Spinner color="green.500" size="large" />
</HStack>`} />
