# IconButton

IconButton component that is based on Ficus UI `Button` component.

## Import

```js
import { IconButton } from 'react-native-ficus-ui';
```

## Usage

### Default

<CodeEditor code={`<VStack spacing={10}>

  <IconButton icon={<Icon name="home" />} colorScheme="teal" />

## Props

https://reactnative.dev/docs/button#props

/>

### `colorScheme`
<PropsTable

/>

### `size`
<PropsTable

description="The full property, if true, then the button takes all the width."
