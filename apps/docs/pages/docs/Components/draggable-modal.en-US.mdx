# DraggableModal

Component to render a draggable modal, based on [`react-native-bottom-sheet`](https://github.com/gorhom/react-native-bottom-sheet).
It wraps a `BottomSheetModal` and automatically embeds a `BottomSheetScrollView` so that your content is scrollable out of the box.

## Import

```js
import { DraggableModal } from "react-native-ficus-ui";
```

## Usage

<CodeEditor code={`const TestModal = () => {
  const { isOpen, onOpen, onClose } = useDisclosure();

  return (
    <FicusProvider>
      <Box h={500} bg="gray.50" p="xl">
        <Button
          colorScheme={!isOpen ? 'green' : 'red'}
          onPress={() => {
            if (!isOpen) {
              onOpen();
            } else {
              onClose();
            }
          }}
        >
          {!isOpen ? 'Show Modal' : 'Hide Modal'}
        </Button>

        <DraggableModal
          isOpen={isOpen}
          onClose={onClose}
          snapPoints={['30%', '80%']}
          p="xl"
        >
          <Text fontSize="4xl" fontWeight="bold">
            Settings
          </Text>

          <Text mt="xl">Your settings</Text>
        </DraggableModal>
      </Box>
    </FicusProvider>
  );
}
render(<TestModal />)`} noInline />
