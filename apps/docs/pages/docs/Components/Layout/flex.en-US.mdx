# Flex

As `Box` component but with `flex={1}` property by default, it accepts every props from react native `View` component.

## Import

```js
import { Flex } from "react-native-ficus-ui";
```

## Usage

<CodeEditor code={`<Box h={200}>
  <Flex bg="red.500" />
</Box>`} />

## Props

Extends every `Box` props.
