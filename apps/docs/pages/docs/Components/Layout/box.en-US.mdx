# Box

Wrapper around `View` component from `react-native`, it accepts every props from react native `View` component.

## Import

```js
import { Box } from 'react-native-ficus-ui';
```

## Usage

### Background color

<CodeEditor code={`<HStack spacing={10}>
  <Box h={40} w={40} bg="green.500" />
  <Box h={40} w={40} bg="teal.500" />
  <Box h={40} w={40} bg="yellow.500" />
  <Box h={40} w={40} bg="red.500" />
  <Box h={40} w={40} bg="blue.500" />
  <Box h={40} w={40} bg="gray.500" />
</HStack>`} />

### Shadows

<CodeEditor code={`<HStack spacing={10}>
  {['none', 'xs', 'sm', 'md', 'lg', 'xl', '2xl'].map(shadow => (
    <Box
      h={40}
      w={40}
      bg="white"
      borderRadius="sm"
      shadow={shadow}
      justifyContent="center"
      alignItems="center"
    >
      <Text>{shadow}</Text>
    </Box>
  ))}
</HStack>`} />

<Callout type="info" emoji="ðŸŽ‰">
  React Native 0.76 introduces new style props `boxShadow` and `filter` for `View`, those props are now available on Ficus UI from version 1.3.0.
  You can pass `boxShadow` and `filter` props directly to `Box` component.
</Callout>

https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture#box-shadow-and-filter-style-props

### As prop

The as prop can be used to style any React Native component with Ficus UI.

<CodeEditor code={`<Box
  as={ReactNative.TouchableOpacity}
  bg="green.400"
  p="xl"
  mt="sm"
  borderRadius="md"
>
  <Text>Style any RN component with Ficus UI with "as" prop !</Text>
</Box>`} />
