# PinInput

Component that is based on `react-native-confirmation-code-field` library.

https://github.com/retyui/react-native-confirmation-code-field

## Import

```js
import { PinInput } from 'react-native-ficus-ui';
```

## Usage

### Default

<CodeEditor code={`const SimplePinInput = () => {
  const [pinValue, setPinValue] = React.useState(null);

  return (
    <PinInput
      value={pinValue}
      onChangeText={setPinValue}
      keyboardType="number-pad"
      w={250}
    />
  );
}
render(<SimplePinInput />)`} noInline />
