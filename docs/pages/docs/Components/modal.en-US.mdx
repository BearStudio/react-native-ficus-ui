# Modal

Component to render a modal imported from `react-native-modal`

https://github.com/react-native-modal/react-native-modal

## Import

```js
import { Modal } from 'react-native-ficus-ui';
```

## Usage

<div className="mt-2">
  <ExpoDemo id="@bearstudio/ficus-ui-modal" />
</div>
