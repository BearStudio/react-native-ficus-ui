# HStack

HStack is a layout component to display several children elements horizontally with a space between them.

## Import

```js
import { HStack } from 'react-native-ficus-ui';
```

## Usage

<div className="mt-2">
  <ExpoDemo id="@bearstudio/ficus-ui-stack" />
</div>
