# Icon

Component to display icon, it's a wrapper around the `Icon` component from `react-native-vector-icons` library.

## Import

```js
import { Icon } from 'react-native-ficus-ui';
```

## Usage

<Box mt={5}>
  <ExpoDemo id="@bearstudio/ficus-ui-icon" />
</Box>
